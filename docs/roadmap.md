# FluxBin Roadmap

这份文档不再单独维护详细阶段内容。

当前已经把：

- 当前状态
- 已完成能力
- 当前缺口
- 下一步完整计划

统一收敛到：

- [docs/overview/current-gaps.md](./overview/current-gaps.md)

## 为什么收敛

之前 `roadmap` 和 `current-gaps` 同时承载：

- 现状描述
- 迁移历史
- 下一步计划

会带来两个问题：

1. 同一事实在两个文档里重复维护
2. “当前状态”和“目标态”容易混写

所以从现在开始：

- `docs/overview/current-gaps.md` 是当前阶段的唯一事实源
- `roadmap.md` 仅保留为入口和兼容跳转页

## 使用方式

如果你要了解：

- 现在做到了哪里
- 还缺什么
- 下一步怎么迭代

直接看：

- [Current Status And Next Steps](./overview/current-gaps.md)
